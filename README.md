StudyHub - E-Learning & Tutor Matching Platform: Giới thiệu ngắn gọn.

Key Features:

Advanced Matching Algorithm: Tự động kết nối gia sư và học viên dựa trên tọa độ địa lý (latitude, longitude) và tính cách (Personality Score).

eKYC Integration: Hệ thống xác thực danh tính gia sư bằng Face Matching và kiểm tra CCCD.

Escrow Payment: Hệ thống thanh toán trung gian an toàn, giữ tiền và giải ngân dựa trên trạng thái hợp đồng.

Smart Address Extraction: Tự động chuẩn hóa địa chỉ đầu vào thành các thành phần chi tiết (Alley, Street, Ward,...) 

Tech Stack: Spring Boot, React (Vite), MS SQL Server, Spring Security (JWT), Hibernate/JPA.
